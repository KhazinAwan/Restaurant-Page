# Restaurant Page

A simple restaurant homepage built as part of **The Odin Project** curriculum. This project focuses on practicing **DOM manipulation**, **ES6 modules**, and **Webpack** by dynamically generating all page content with JavaScript.

## Features

* Single-page restaurant website
* Dynamic tab navigation (Home, Menu, Contact)
* All page content generated using JavaScript
* Modular code structure using ES6 modules
* Styled with vanilla CSS
* Bundled with Webpack

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6 Modules)
* Webpack

## Learning Objectives

This project demonstrates:

* DOM manipulation
* Dynamic page rendering
* Event-driven tab navigation
* ES6 module imports and exports
* Webpack configuration and bundling
* Organizing code using a modular project structure
* Separation of concerns through modular design

## Future Improvements

* Add responsive layouts for different screen sizes
* Include product images and branding assets
* Add animations and transitions for tab switching
* Improve accessibility with keyboard navigation and ARIA attributes
